# 电报有几个数据中心

-----------------------

什么是数据中心，简单来说，就是存储你账号数据的服务器的，而电报存储账号数据的服务器有五个，分别是DC1、DC2、DC3、DC4、DC5，由于DC3已经作废了，故此剩下的电报数据中心就剩下四个

-----------------------

DC1：

> DC1代表的数据中心所在地是美国的迈阿密

DC2：

> DC2代表的数据中心所在地是荷兰的阿姆斯特丹

DC3：(该数据中心已作废)

> DC3代表的数据中心所在地是美国的迈阿密

DC4：

> DC4代表的数据中心所在地是荷兰的阿姆斯特丹

DC5:

> DC5代表的数据中心所在地是新加坡
